# Zinciri-Kirma-Local

> Tarayıcı tabanlı alışkanlık takip uygulaması. Zinciri kırma metodolojisi, joker günler, rozetler ve ısı haritası ile 2026-2035 arası yıllık takvim.
> Her gün bir halka — zincir büyüsün.
> Tarayıcı tabanlı, sıfır bağımlılıklı alışkanlık takip uygulaması. Jerry Seinfeld'in "Don't Break the Chain" metodolojisine dayanır.

## Canlı Demo

> [EmirhanSariyar.github.io/zinciri-kirma](https://EmirhanSariyar.github.io/zinciri-kirma)

## Özellikler

- **Yıllık Takvim** — 2026–2035 arası, artık yıl destekli
- **Çoklu Hedef** — Her hedef için ayrı renk ve takip
- **Joker Günler** — Seriyi kırmadan 1 gün pas geçme hakkı (ardışık 2 joker yasak)
- **Rozetler** — 3, 7, 14, 21, 30, 50, 100, 365 günlük seri rozetleri
- **Isı Haritası** — GitHub tarzı, aktiviteye göre renk yoğunluğu
- **İstatistikler** — Mevcut seri, en uzun seri, tamamlanma oranı
- **Motivasyon Sözleri** — 21 özlü söz, yenilenebilir
- **Otomatik Kayıt** — Tüm veriler tarayıcıda (localStorage) saklanır

## Kurulum

Hiç kurulum gerekmez. Tek dosya:
```bash
git clone https://github.com/kullaniciadin/zinciri-kirma.git
cd zinciri-kirma
# ZinciriKirma.html dosyasını tarayıcıda aç
```

Ya da direkt [indirip](ZinciriKirma.html) tarayıcıda çalıştırabilirsin.

## Teknoloji

- Vanilla HTML / CSS / JavaScript
- Sıfır framework, sıfır bağımlılık
- Google Fonts (Playfair Display, DM Sans, DM Mono)
- localStorage ile veri saklama

## Nasıl Kullanılır?

1. Sol panelden **yeni hedef ekle** (renk seç, isim ver)
2. Hedefi seçili haldeyken takvimde **tamamladığın günlere tıkla**
3. Seriyi kırmamak için günlük **joker** kullanabilirsin
4. İstatistik sekmesinde ilerlemeyi ve rozetleri takip et
